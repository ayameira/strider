====================
# Backend
cd backend

# Edit .env with your OpenAI API key

# iOS
open StrideMind/StrideMind.xcodeproj
# Press Cmd+R in Xcode to run


API Endpoints
=============
POST /api/plan/generate      - Generate training plan
POST /api/workout/analyze    - Analyze completed workout
POST /api/plan/adjust        - Adjust training plan
POST /api/coach/chat         - Chat with AI coach
GET  /health                 - Health check
